# C Programming Basics Quiz
Created: 2025-08-07 21:18:11 IST

## Questions

1. What is the size of int data type in C (on a typical 32-bit system)?
   - A) 2 bytes
   - B) 4 bytes
   - C) 8 bytes
   - D) 1 byte
   Correct: B

2. Which operator is used for pointer declaration?
   - A) &
   - B) *
   - C) #
   - D) @
   Correct: B

3. What is the correct way to declare a function pointer?
   - A) void (*fp)();
   - B) void *fp();
   - C) void fp*();
   - D) void fp();
   Correct: A

4. What happens if malloc() fails to allocate memory?
   - A) Returns -1
   - B) Returns NULL
   - C) Returns void
   - D) Program crashes
   Correct: B

5. Which header file is required for dynamic memory allocation?
   - A) <string.h>
   - B) <stdlib.h>
   - C) <memory.h>
   - D) <alloc.h>
   Correct: B
