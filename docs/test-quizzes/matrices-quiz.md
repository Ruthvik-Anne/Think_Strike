Created: 2025-08-07 21:18:11 IST

## Questions

1. For a 3x3 matrix, what is the order of its cofactor matrix?
   - A) 2x2
   - B) 3x3
   - C) 3x2
   - D) 2x3
   Correct: B

2. What is the determinant of a singular matrix?
   - A) 1
   - B) -1
   - C) 0
   - D) Undefined
   Correct: C

3. For matrix multiplication to be possible, what condition must be met?
   - A) Matrices must be square
   - B) Columns of first = Rows of second
   - C) Rows of first = Columns of second
   - D) Both matrices must be same size
   Correct: B

4. What is the rank of a zero matrix of size 4x4?
   - A) 0
   - B) 1
   - C) 4
   - D) 16
   Correct: A

5. If A is invertible, then |A⁻¹| equals:
   - A) |A|
   - B) 1/|A|
   - C) -|A|
   - D) 0
   Correct: B
