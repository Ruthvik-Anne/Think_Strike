## Questions

1. What is the Nyquist sampling rate for a 4 kHz signal?
   - A) 4 kHz
   - B) 8 kHz
   - C) 12 kHz
   - D) 16 kHz
   Correct: B

2. Which modulation technique is most resistant to noise?
   - A) AM
   - B) FM
   - C) PM
   - D) ASK
   Correct: B

3. What is the bandwidth required for a baseband binary signal with bit rate R?
   - A) R/2 Hz
   - B) R Hz
   - C) 2R Hz
   - D) 4R Hz
   Correct: B

4. In QPSK modulation, how many bits are transmitted per symbol?
   - A) 1
   - B) 2
   - C) 4
   - D) 8
   Correct: B

5. What is the purpose of Gray coding in digital modulation?
   - A) Error detection
   - B) Minimize bit errors
   - C) Increase bandwidth
   - D) Reduce power consumption
   Correct: B
